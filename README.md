# BRRRR Investment Automation System

Complete automation system for managing BRRRR (Buy, Rehab, Rent, Refinance, Repeat) real estate investments in Acworth/Cartersville, Georgia.

## System Components

### 1. Deal Tracker Database (Notion)
- Track properties through the complete BRRRR cycle
- 40+ fields for comprehensive deal analysis
- Auto-calculated financial metrics
- **URL:** https://www.notion.so/936926d8f35342e1a43c4bb66fc04e34

### 2. Market Analysis Tracker Database (Notion)
- Understand local market conditions
- 50+ fields for market research
- Neighborhood comparables database
- **URL:** https://www.notion.so/2e33ce7a0b4f49d693d8c76b4acec2ea

### 3. Python Automation Script
- `brrrr_market_automation.py` - Automates market data collection
- Pulls data from Zillow, county assessor records
- Auto-creates Notion entries
- Scheduled monthly updates
